# matrices_sum
В папке openCL_pr вы найдете все, что необходимо для работы этого проекта.

### build & run
Находясь в папке проекта:
1. удалить существующую build-директорию (если есть)
   ```bash
   rm -rf build
2. создать новую build-директорию
   ```bash
   mkdir build
   cd build
   cmake ..
   make
3. запустить тесты
   ```bash
   ./tests
Profit!
